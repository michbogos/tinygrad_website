Tinygrad is a simple, yet powerful neural network framework. It's extremely simple and breaks down the most complex networks into 4 OpTypes.

API similar to pytorch

The tinygrad API is very close to pytorch. You can start with tinygrad without the learning curve.

4 optypes

Tinygrad can represent even the most complex neural networks with just 4 ops. This makes the simple to reason about. Porting new hardware is also much easier.

Very low dependencies

The code is self-contained. The only dependencies are numpy and tqdm.

JIT compilation

The kernels are generated and compiled on the fly. This means higher performance on a wider range of hardware.

#Install tinygrad
python3 -m pip install git+https://github.com/tinygrad/tinygrad.git

from tinygrad import Tensor

x = Tensor.eye(3, requires_grad=True)
y = Tensor([[2.0,0,-2.0]], requires_grad=True)
z = y.matmul(x).sum()
z.backward()

print(x.grad.numpy())  # dz/dx
print(y.grad.numpy())  # dz/dy
